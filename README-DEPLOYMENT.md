# Project India Deployment

Production dashboard:

```text
https://project-india-nflujcnhq3f7xfj2d6q6sh.streamlit.app/
```

The Streamlit app is the public presentation layer for Project India. It starts from an empty topic library and lets the admin submit new topics through the **Start Research** page.

## Streamlit Cloud

Use these settings:

```text
Repository: knirantar/Project-India
Branch: main
Main file path: dashboard.py
Python: 3.11 or 3.12
```

## Streamlit Secrets

Add these in Streamlit Cloud app settings:

```toml
GITHUB_OWNER = "knirantar"
GITHUB_REPO = "Project-India"
GITHUB_DISPATCH_TOKEN = "github_token_with_actions_write"
APP_ADMIN_PIN = "private-admin-pin"
```

## GitHub Actions Secrets

Required:

```text
OPENAI_API_KEY
```

Optional, for automatic PR merge after research:

```text
PROJECT_INDIA_ADMIN_TOKEN
```

## Flow

```text
Start Research form
-> topic-intake-research.yml
-> source-backed OpenAI research
-> topic note, source log, brief, structured topic data, run record, research index
-> PR into main
-> Streamlit redeploys from main
```

No PPTX deck is generated by this app-first workflow.
